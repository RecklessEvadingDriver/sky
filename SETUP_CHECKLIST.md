# Theme Setup Checklist

Use this checklist to ensure your Skystar Tour and Travels theme is properly configured.

## Pre-Installation
- [ ] Have Shopify store access
- [ ] Have theme files ready to upload
- [ ] Reviewed INSTALLATION.md guide

## Installation
- [ ] Uploaded theme to Shopify
- [ ] Theme appears in Online Store → Themes

## Collections Setup
- [ ] Created `domestic-tours` collection
- [ ] Created `international-tours` collection
- [ ] Created `honeymoon-packages` collection
- [ ] Created `popular-packages` collection
- [ ] Added at least 1 product to each collection

## Theme Settings Configuration
- [ ] Updated WhatsApp number (format: 919999999999)
- [ ] Set contact phone number
- [ ] Set contact email address
- [ ] Set contact physical address
- [ ] Updated contact hours (if needed)
- [ ] Uploaded favicon
- [ ] Uploaded OG image for social sharing
- [ ] Added social media URLs (Facebook, Instagram, etc.)

## Metafields Setup
- [ ] Created `tour.duration` metafield (single line text)
- [ ] Created `tour.price_label` metafield (single line text)
- [ ] Created `tour.inclusions` metafield (single line text)
- [ ] Created `tour.group_size` metafield (single line text)
- [ ] Created `tour.best_time` metafield (single line text)
- [ ] Created `tour.itinerary` metafield (multi-line text)

## Product Setup
- [ ] Created at least 3-5 tour package products
- [ ] Added product images (high quality)
- [ ] Set product prices
- [ ] Assigned products to correct collections
- [ ] Applied `product.skystar` template to products
- [ ] Filled in metafields for each product

## Homepage Configuration
- [ ] Verified hero section displays correctly
- [ ] Verified trust strip shows stats
- [ ] Verified featured collections section works
- [ ] Verified popular packages section displays products
- [ ] Verified features section is visible
- [ ] Verified testimonials section appears
- [ ] Verified contact CTA section works

## Testing
- [ ] Tested homepage on desktop
- [ ] Tested homepage on mobile
- [ ] Tested product page on desktop
- [ ] Tested product page on mobile
- [ ] Tested collection pages
- [ ] Verified WhatsApp floating button works
- [ ] Verified WhatsApp inquiry buttons work
- [ ] Tested header navigation
- [ ] Tested footer links
- [ ] Tested mobile menu
- [ ] Verified product recommendations appear
- [ ] Verified scroll-to-top button works
- [ ] Tested all internal links

## Performance & SEO
- [ ] Checked page load speed
- [ ] Verified all images load properly
- [ ] Checked meta descriptions
- [ ] Verified favicon appears in browser tab
- [ ] Tested social media preview (share on Facebook/WhatsApp)
- [ ] Verified structured data (check with Google Rich Results Test)

## Final Steps
- [ ] Published theme (if not already)
- [ ] Tested theme in incognito/private browsing mode
- [ ] Reviewed entire site for any issues
- [ ] Created backup of theme

## Go Live!
- [ ] Theme is live and working properly
- [ ] Monitoring for any customer-reported issues
